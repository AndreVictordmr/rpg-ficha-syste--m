# RPG Sheet System
## Sobre o projeto

Sistema de criação e gerenciamento de fichas para RPG e jogos em geral que seguem essa temática. O projeto foi desenvolvido com foco em práticas de desenvolvimento back-end, estruturação de sistemas e evolução de habilidades em programação.

Também tem como objetivo o aprendizado e aplicação de novas tecnologias, incluindo TypeScript e Node.js.
